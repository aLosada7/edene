## Import

```bash
import { useHover } from '@edene/components';
```

Detects if an element is been hovered

## Usage

useHover can be useful to show a tooltip.

```tsx
() => {
    const ref = React.useRef(null);
    const [hoverRef, hovering] = useHover(ref);

    return (
        <div ref={hoverRef}>
            <Text>Hover me!</Text>
            {hovering && <Tooltip />}
        </div>
    );
};
```
