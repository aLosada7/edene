## Import

```bash
import { Image } from '@edene/components';
```

## About

-   Uses **loading=lazy** to prevent downloading off-screen images

## Usage

```tsx
<Image
    src="https://images.pexels.com/photos/13248572/pexels-photo-13248572.jpeg"
    alt="Tower"
/>
```
