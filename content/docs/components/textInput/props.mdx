## TextInput

| Name     | Type    | Description                                                 |
| :------- | :------ | :---------------------------------------------------------- |
| label    | string  | Input label, displayed before the input                     |
| error    | string  | Displays an error message after the input                   |
| success  | string  | Displays an success message after the input                 |
| disabled | boolean | Disabled input state                                        |
| iconLeft | string  | Displays the value as an icon at the left side of the input |
