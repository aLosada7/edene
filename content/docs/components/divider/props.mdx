## Badge

| Name      | Type   | Description                                                   |
| :-------- | :----- | :------------------------------------------------------------ |
| component | 'li'   | Polymorphic component                                         |
| width     | number | Specific width. If missing, default is _0_, which equals 100% |
| mt        | number | Margin top                                                    |
| mb        | number | Margin bottom                                                 |
| mh        | number | Horizontal margin                                             |
