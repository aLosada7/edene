## Text

| Name     | Type                                                                | Description                           |
| :------- | :------------------------------------------------------------------ | :------------------------------------ |
| mt       | Scale                                                               | Each unit is multipled by 0.25 rem    |
| mb       | Scale                                                               | Each unit is multipled by 0.25 rem    |
| size     | 'xxsmall' \| 'xsmall' \| 'small' \| 'md' \| 'lg' \| 'xlg' \| 'xxlg' | Size of text content. Default is _md_ |
| children | ReactElement \| string                                              | Text for the elements                 |
