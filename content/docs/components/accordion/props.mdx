## Accordion

| Name            | Type                       | Description                             |
| :-------------- | :------------------------- | :-------------------------------------- |
| children        | ReactElement               | _AccordionRow_ components only          |
| hideToggleLabel | boolean                    | Hidden _Show_ and _Hide_ labels         |
| variant         | 'borderless' \| 'bordered' | Accordion variant. Default _borderless_ |

## AccordionRow

| Name     | Type      | Description                          |
| :------- | :-------- | :----------------------------------- |
| children | ReactNode | Content of accordion row when opened |
| label    | string    | Heading text for the accordion row   |
