## Tabs

| Name        | Type                       | Description                                        |
| :---------- | :------------------------- | :------------------------------------------------- |
| children    | ReactElement               | _Tab_ components only                              |
| active      | boolean                    | The selected tab value                             |
| onTabChange | callback                   | Callback when a tab has been selected              |
| orientation | 'vertical' \| 'horizontal' | Defines the tabs orientation. Default \_horizontal |

## Tab

| Name     | Type      | Description                          |
| :------- | :-------- | :----------------------------------- |
| children | ReactNode | Content of accordion row when opened |
| label    | string    | Heading text for the tab head        |
| disabled | boolean   | If the tab is disabled               |
| hidden   | boolean   | If the tab is disabled               |
