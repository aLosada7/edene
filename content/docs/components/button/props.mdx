## Button

| Name      | Type                                                | Description                                                                                |
| :-------- | :-------------------------------------------------- | :----------------------------------------------------------------------------------------- |
| children  | ReactElement                                        | A _React_ component only                                                                   |
| variant   | 'filled' \| 'outline' \| 'link'                     | Button variant. Default _filled_                                                           |
| size      | 'xsmall' \| 'small' \| 'medium' \| 'large'          | Button size. Default _medium_                                                              |
| iconLeft  | string                                              | Icon to show at the left of the text. From [Google Fonts](https://fonts.google.com/icons)  |
| iconRight | string                                              | Icon to show at the right of the text. From [Google Fonts](https://fonts.google.com/icons) |
| iconSize  | 'xsmall' \| 'small' \| 'medium' \| 'large' \| 'big' | Size of the icon. Only taken on account if _iconLeft_ or _iconRight_ exists                |
| disabled  | boolean                                             | If the button is disabled                                                                  |
| ref       | Ref                                                 | Button ref                                                                                 |

## ButtonGroup

| Name     | Type                                               | Description                                                                                  |
| :------- | :------------------------------------------------- | :------------------------------------------------------------------------------------------- |
| children | ReactElement                                       | A _React_ component only                                                                     |
| variant  | 'filled' \| 'outline' \| 'link'                    | Button Group variant. Shows all buttons in the group with the same variant. Default _filled_ |
| size     | 'xsmall' \| 'small' \| 'medium' \| 'large' \|block | Button size (block to ocupy entire width). Default _medium_                                  |
