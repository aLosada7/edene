## Import

```bash
import { FadeIn } from '@edene/animations';
```

## Usage

```tsx
<FadeIn>
    <Text type="regular">
        Present your card to a newsagent each time you collect the paper. The
        newsagent will scan your card and will be reimbursed for each
        transaction automatically.
    </Text>
</FadeIn>
```
