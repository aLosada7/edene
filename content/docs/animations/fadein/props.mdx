## Accordion

| Name     | Type                           | Description                                   |
| :------- | :----------------------------- | :-------------------------------------------- |
| children | ReactElement \| ReactElement[] | Content you want to be animated               |
| delay    | string                         | Time to be delay in seconds. Default is _.5s_ |
| variant  | 'borderless' \| 'bordered'     | Accordion variant. Default _borderless_       |
