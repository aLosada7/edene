# Icons

## Usage

Transitions are exported from foundations

## List of available animations

### Short

<div
    id="transition-short"
    style={{
        width: '100px',
        height: '100px',
        backgroundColor: 'red',
        transition: 'transform ' + transitions.short,
    }}
>
    {' '}
</div>
<button
    style={{ margin: '1rem 0 2rem' }}
    onClick={() => {
        document.getElementById('transition-short').style.transform =

</button>
