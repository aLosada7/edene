# useHover

The useHover hook allows the programmer to know when a DOM element is being hovered

## Example

In this example the screen displays a tooltip when the Text is being hovered

### Code

```tsx
() => {
    const ref = useRef<HTMLDivElement>(null);
    const [hoverRef, hovering] = useHover(ref);

    return (
        <div ref={hoverRef}>
            <Text type="regular">Hover me!</Text>
            {hovering && <Tooltip />}
        </div>
    );
};
```
