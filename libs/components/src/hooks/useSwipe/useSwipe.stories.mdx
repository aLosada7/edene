# useSwipe

The useSwipe hook enables swipping right and left in both desktop and mobile devices

## Example

In this example you can see a text showing if you have swipping right or left

### Code

```tsx
() => {
    const ref = useRef<HTMLDivElement>(null);
    const [scrollRef, swipe] = useSwipe(ref);

    return (
        <div ref={scrollRef}>
            <Text type="regular">You swiped {swipe}</Text>
        </div>
    );
};
```
